# Simple Console-Based Games and Calculator

This repository contains three simple console-based programs written in C++:

1. **Tic-Tac-Toe Game**: A two-player game.
2. **Calculator**: A basic calculator that performs arithmetic operations.
3. **Number Guessing Game**: A game where the user guesses a randomly generated number.

## Tic-Tac-Toe Game

### Features
- 3x3 game board.
- Two players: 'X' and 'O'.
- Displays the current state of the board.
- Prompts players to enter their moves.
- Updates the board with the players' moves.
- Checks for win or draw conditions.
- Switches turns between players.
- Displays the result of the game (win, draw, or ongoing).
- Option to play again.
